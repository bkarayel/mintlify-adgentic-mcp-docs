## Deployment Details

- **MCP Endpoint:** `https://mcp.adgenticplatform.com/mcp/`  
- **Transport:** MCP over HTTP (streamable-http), JSON-RPC 2.0 + SSE  
- **Headers:** `Content-Type: application/json`, `Accept: application/json, text/event-stream`
