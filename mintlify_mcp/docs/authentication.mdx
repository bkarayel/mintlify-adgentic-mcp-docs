<div style={{ display:'flex', flexDirection:'column', gap:'16px' }}>
  <a href="https://publisher.adgenticplatform.com/" target="_blank" style={{ textDecoration:'none', display:'block' }}>
    <div style={{ border:'1px solid var(--mint-c-border)', borderRadius:'12px', padding:'16px' }}>
      <h4 style={{ marginTop:0 }}>1) Sign up on Adgentic Publisher Platform</h4>
      <p>Create an account to access the dashboard.</p>
    </div>
  </a>

  <a href="https://publisher.adgenticplatform.com/" target="_blank" style={{ textDecoration:'none', display:'block' }}>
    <div style={{ border:'1px solid var(--mint-c-border)', borderRadius:'12px', padding:'16px' }}>
      <h4 style={{ marginTop:0 }}>2) Obtain your ADGENTIC_API_KEY</h4>
      <p>Go to <em>API Keys</em> in the dashboard and create a new key.</p>
    </div>
  </a>

  <div style={{ border:'1px solid var(--mint-c-border)', borderRadius:'12px', padding:'16px' }}>
    <h4 style={{ marginTop:0 }}>3) Provide your <code>ADGENTIC_API_KEY</code> either:</h4>
    <ul>
      <li>Per call: <code>"api_key": "&lt;YOUR_KEY&gt;"</code></li>
      <li>
        <p>Or as an environment variable:</p>

        <p><strong>macOS / Linux (bash, zsh)</strong></p>
        <pre><code>export ADGENTIC_API_KEY="&lt;YOUR_KEY&gt;"</code></pre>

        <p><strong>Windows (CMD)</strong></p>
        <pre><code>set ADGENTIC_API_KEY="&lt;YOUR_KEY&gt;"</code></pre>

        <p><strong>Windows (PowerShell)</strong></p>
        <pre><code>$Env:ADGENTIC_API_KEY="&lt;YOUR_KEY&gt;"</code></pre>
      </li>
    </ul>
  </div>
</div>
