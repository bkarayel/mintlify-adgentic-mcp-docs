**Tool Name:** `match_adgentic_products`  
Searches and returns structured offers based on the user’s prompt and optional context.

**Parameters:**
- `prompt` *(string, required)* → User query, e.g., `"I am training for a marathon and I need a waterproof smartwatch under $300"`
- `api_key` *(string, required unless set in env)*
- `context` *(object, optional)* → Location/Country, e.g., `{ "country": "US" }`
